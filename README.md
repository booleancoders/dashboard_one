# Pulse Analytics Dashboard

![Pulse Dashboard](https://img.shields.io/badge/UI-Light_Mode-F0E8D5?style=flat-square&logo=design)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)

A hyper-premium, fully responsive, and highly polished modern analytics dashboard template built exclusively with vanilla HTML and CSS. The design has been meticulously tuned for an ultra-aesthetic Dribbble presentation, featuring a warm, light-mode color theme utilizing off-whites, beiges, and deep warm browns.

## ✨ Features

- **Pristine Aesthetics:** Modern, "glassy" and sleek light-mode UI inspired by the best Dribbble SaaS concepts.
- **Warm Color Palette:** 
  - Backgrounds: `#F0E8D5` (Warm off-white) & `#FCFAF6` (Beige surfaces)
  - Layout Accents: `#443226`, `#735D4F`, & `#A67D65` (Warm dark-to-light browns)
- **High-Quality Typography:** Beautifully implemented `Montserrat` font from Google Fonts for a clean, legible, and premium appearance.
- **Data Visualization Elements:**
  - KPI Cards with integrated SVGs and Sparklines
  - Beautifully plotted Revenue Line Chart with gradient fills
  - Segmented Traffic Sources Donut Chart
- **Flexible Grid Layout:** Organized CSS Grid and Flexbox integrations for effortless scalability.
- **Organized Codebase:** Cleanly separated `index.html` structure and modular `pulse-dashboard-web.css`.

## 📂 File Structure

```text
├── pulse-dashboard-web.html     # Main structural HTML containing markup for sidebar, header, charts, and tables
├── pulse-dashboard-web.css      # Separated stylesheet governing layout, themes, properties, and colors
└── README.md                    # Project documentation
```

## 🚀 Getting Started

No installations or build procedures are required. 
1. Clone this repository to your local machine.
2. Open `pulse-dashboard-web.html` in any modern web browser to view the functional UI.

## 🎨 Design System Overview

- **Font Family:** `Montserrat`, sans-serif
- **Primary Text:** `#291F1A` (Deep dark brown for maximum readable contrast)
- **Primary Accent:** `#443226`
- **Secondary Accent:** `#735D4F`
- **Borders & Separators:** `rgba(41, 31, 26, 0.1)`

## 💡 Usage

This repository is perfect as a starting point for integrating backend frameworks (React, Vue, Next.js, etc.). All UI elements are standard HTML/CSS, meaning they are completely framework agnostic and incredibly fast out-of-the-box.
